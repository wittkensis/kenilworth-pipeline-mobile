# Blockers

Agents: append here immediately when hitting a blocker. Also create a GitHub Issue with label `blocker`.

## Format

```
## BLOCKER — {Date} — {Brief description}

**What:** {What you were trying to do}
**Blocked by:** {Specific error, constraint, or unknown}
**Impact:** {What can't proceed until resolved}
**Resolution:** Pending / Resolved on {date}: {how}
```
